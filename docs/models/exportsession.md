# ExportSession

Comprehensive session export with all messages, tool calls, and metadata

## Example Usage

```typescript
import { ExportSession } from "mix-typescript-sdk/models";

let value: ExportSession = {
  id: "<id>",
  messages: [
    {
      content: "<value>",

      id: "<id>",
      role: "<value>",
      updatedAt: new Date("2023-08-18T22:35:02.040Z"),
    },
  ],
  title: "<value>",
};
```
